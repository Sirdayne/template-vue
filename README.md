# template vue

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for development with minification and zip file packaging
npm run build-dev

# build for QA with minification and zip file packaging
npm run build-qa

# build for production with minification and zip file packaging
npm run build-prod

# build for production and view the bundle analyzer report
npm run build-prod --report
```
